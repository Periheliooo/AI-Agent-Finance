  项目结构总结：

  ┌─────────────────────────┬──────────────────────────────────────────┐
  │          文件           │                   职责                   │
  ├─────────────────────────┼──────────────────────────────────────────┤
  │ config.py               │ 所有常量（初始价格、alpha、轮数等）      │
  ├─────────────────────────┼──────────────────────────────────────────┤
  │ market.py               │ 市场类，维护价格 + 根据买卖单更新        │
  ├─────────────────────────┼──────────────────────────────────────────┤
  │ llm.py                  │ LLM 接口占位，generate_reason() 返回空串 │
  ├─────────────────────────┼──────────────────────────────────────────┤
  │ agents/base_agent.py    │ Agent 基类：现金、持仓、memory、act 方法 │
  ├─────────────────────────┼──────────────────────────────────────────┤
  │ agents/fomo_agent.py    │ 追涨杀跌：连续涨→BUY，连续跌→SELL        │
  ├─────────────────────────┼──────────────────────────────────────────┤
  │ agents/value_agent.py   │ 价值投资：低于均线→BUY，高于均线→SELL    │
  ├─────────────────────────┼──────────────────────────────────────────┤
  │ agents/gambler_agent.py │ 赌徒：40% BUY / 40% SELL / 20% HOLD      │
  ├─────────────────────────┼──────────────────────────────────────────┤
  │ simulation.py           │ 主循环：每轮驱动决策→更新价格→输出日志   │
  └─────────────────────────┴──────────────────────────────────────────┘
